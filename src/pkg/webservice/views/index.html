  • Mogelijke calls
    • GET /article/category/{category_id} (Alle artikelen per category [1..n])
    • GET /article (Alle artikelen)
    • GET /article/{article_id} (1 artikel met id [1..n])
    • GET /category (Retourneer alle hoofdcategorien)
    • POST /comment (Maak een comment aan POST waarden: post_id, name, email en content)
    • GET /log (Laat de log zien)
    • DELETE /log (Leeg de log)
    • GET /search/{zoek_waarde} (Zoek naar een waarde)
